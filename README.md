## ✨ Qué incluye
- Diseño responsive con **TailwindCSS**
- Secciones: **Inicio**, **Sobre mí**, **Habilidades**, **Proyectos**, **Contacto**
- Tarjetas y carrusel de **proyectos** con imágenes
- Animaciones sutiles (**AOS**)

## 🧰 Stack
- **React 18**, **Vite**
- **TailwindCSS**
- Icons/imagenes optimizadas (WebP)
